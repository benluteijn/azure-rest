              SqlManagementClient

              Creates or updates an Azure SQL Server security alert policy.

              Request

              PUT /subscriptions/{subscriptionId}/resourceGroups/{resourceGroupName}/providers/Microsoft.Sql/servers/{serverName}/securityAlertPolicies/Default?api-version

              Properties

                1. parameters (required)
                The required parameters for creating or updating a Azure SQL Server security alert policy.

                view child properties

                  1. properties (required)
                  Properties of the request.
                1. resourceGroupName (required)
                2. string
                The name of the Resource Group to which the server belongs.
                1. serverName (required)
                2. string
                The name of the Azure SQL Server.
                1. api-version (required)
                2. string
                Client Api Version.
                1. subscriptionId (required)
                2. string
                The subscription credentials which uniquely identify Microsoft Azure subscription.

              Returns

              HTTP RESPONSE CODES
                1. 200
                The Server Threat Detection Policy
                1. 201
