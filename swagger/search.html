              Returns the primary and secondary API keys for the given Azure Search service.

              Request

              POST /subscriptions/{subscriptionId}/resourceGroups/{resourceGroupName}/providers/Microsoft.Search/searchServices/{serviceName}/listAdminKeys?api-version

              Properties

                1. resourceGroupName (required)
                2. string
                The name of the resource group within the current subscription.
                1. serviceName (required)
                2. string
                The name of the Search service for which to list admin keys.
                1. api-version (required)
                2. string
                Client Api Version.
                1. subscriptionId (required)
                2. string
                Gets subscription credentials which uniquely identify Microsoft Azure subscription. The subscription ID forms part of the URI for every service call.

              Returns

                1. 200
