              NotificationHubsManagementClient

              Checks the availability of the given service namespace across all Windows Azure subscriptions. This is useful because the domain name is created based on the service namespace name.

              Request

              POST /subscriptions/{subscriptionId}/providers/Microsoft.NotificationHubs/checkNamespaceAvailability?api-version

              Properties

                1. parameters (required)
                The namespace name.

                view child properties

                  1. name (required)
                  2. string
                  Gets or sets name
                  1. location
                  2. string
                  Gets or sets location.
                  1. tags
                  2. object
                  Gets or sets tags.
                  1. isAvailiable
                  2. boolean
                  Gets or sets true if the name is available and can be used to create new Namespace/NotificationHub. Otherwise false.
                1. api-version (required)
                2. string
                Client Api Version.
                1. subscriptionId (required)
                2. string
                Gets subscription credentials which uniquely identify Microsoft Azure subscription. The subscription ID forms part of the URI for every service call.

              Returns

              HTTP RESPONSE CODES
                1. 200
