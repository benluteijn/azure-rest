              Gets all job collections under specified subscription.

              Request

              GET /subscriptions/{subscriptionId}/providers/Microsoft.Scheduler/jobCollections?api-version

              Properties

                1. subscriptionId (required)
                2. string
                The subscription id.
                1. api-version (required)
                2. string
                The API version.

              Returns

              HTTP RESPONSE CODES
                1. 200
                The job collections have been successfully returned.

              Lists all jobs under the specified job collection.

              Request

              GET /subscriptions/{subscriptionId}/resourceGroups/{resourceGroupName}/providers/Microsoft.Scheduler/jobCollections/{jobCollectionName}/jobs?api-version[&$top&$skip&$filter]

              Properties

                1. subscriptionId (required)
                2. string
                The subscription id.
                1. resourceGroupName (required)
                2. string
                The resource group name.
                1. jobCollectionName (required)
                2. string
                The job collection name.
                1. api-version (required)
                2. string
                The API version.
                1. $top
                2. integer
                The number of jobs to request, in the of range [1..100].
                1. $skip
                2. integer
                The (0-based) index of the job history list from which to begin requesting entries.
                1. $filter
                2. string
                The filter to apply on the job state.

              Returns

              HTTP RESPONSE CODES
                1. 200
                The job has been successfully returned.
